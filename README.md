# Healthcare Analytics Portfolio (Soledad Banner)

## What this is
A small portfolio of analytics and AI-adjacent case studies focused on healthcare operations, revenue cycle, and documentation/compliance workflows.

## Projects
### 1) Monitoring Revenue by Month (Case Study)
**Goal:** Show monthly volume and revenue patterns for monitoring patients using CPT anchors (e.g., 99000) and patient lists.  
**Methods:** SQL-style logic, Excel/Pivot approach, and a reproducible Python notebook version using synthetic data.  
**Outputs:** Monthly KPI table + chart; example pivot layout used for physician reporting.

### 2) Patient Experience Survey Reporting (Case Study)
**Goal:** Split survey questions into readable executive slides with clean charts and data labels.  
**Methods:** Data cleaning + visualization logic; PowerPoint-ready outputs.  

## Stack
SQL • Excel (Pivots/XLOOKUP) • Python (pandas) • Jupyter

## Notes
All data is synthetic or anonymized. No PHI is included.
